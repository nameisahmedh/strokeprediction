How It Works

Understanding our Machine Learning-Based Stroke Prediction System

What is This System?

This is a Machine Learning-based healthcare tool that analyzes patient data to predict stroke risk. It uses proven statistical algorithms trained on real healthcare datasets.

What It Is
  • Machine Learning classification system
  • Uses 7 proven ML algorithms
  • Trained on real stroke patient data
  • Analyzes 11 health parameters
  • Provides risk predictions
What It's Not
  • Not a medical diagnosis tool
  • Not a replacement for doctors
  • Not generative AI (like ChatGPT)
  • Not a treatment recommendation system
  • Not FDA-approved medical device

The 4-Step Process

1
Upload Data

Provide patient healthcare data in CSV format with 11 required parameters

2
Preprocess

Data is encoded, scaled, balanced with SMOTE, and features are selected

3
Train Model

Choose and train one of 7 ML algorithms on your processed dataset

4
Predict

Get stroke risk predictions for individual patients or batch analysis

7 Machine Learning Algorithms

The system supports multiple proven ML algorithms. You can train and compare all of them:

Random Forest

Ensemble method using multiple decision trees for robust predictions

XGBoost

Gradient boosting algorithm known for high accuracy in classification

Logistic Regression

Statistical model for binary classification with probability outputs

SVM (Support Vector Machine)

Finds optimal decision boundary for classification tasks

K-Nearest Neighbors

Classifies based on similarity to nearest training examples

Decision Tree

Tree-based model making decisions through feature splits

CatBoost

Advanced gradient boosting optimized for categorical features

Data Preprocessing Pipeline

Encoding

Converts categorical data (gender, work_type) to numerical format

Scaling

Normalizes numerical features to the same scale

SMOTE

Balances dataset by creating synthetic minority samples

Feature Selection

Selects most important features using SelectKBest

11 Required Health Parameters

Parameter Type Description Example Values
gender Categorical Patient's gender Male, Female, Other
age Numerical Age in years 0-120
hypertension Binary High blood pressure history 0 (No), 1 (Yes)
heart_disease Binary Heart disease history 0 (No), 1 (Yes)
ever_married Categorical Marital status Yes, No
work_type Categorical Type of occupation Private, Self-employed, Govt_job, children, Never_worked
Residence_type Categorical Living area type Urban, Rural
avg_glucose_level Numerical Average blood glucose (mg/dL) 50-300
bmi Numerical Body Mass Index 10-60
smoking_status Categorical Smoking history formerly smoked, never smoked, smokes, Unknown
stroke Target Stroke occurrence (training only) 0 (No), 1 (Yes)
